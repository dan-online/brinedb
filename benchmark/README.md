# BrineDB Benchmark

To run the benchmark, you need to follow these steps:

1. Run ``docker compose up -d``
2. Run ``cd .. && yarn release-native && cd benchmark``
3. Run ``yarn bench``

The benchmark will run and output the results to the console.
